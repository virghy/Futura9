How to: Display Items in Databases
See Also 






You can choose to which database items to display or how to display them in the Database Designer. For example, you can expand tables to see the fields and indexes they contain or collapse tables to show only table names.

To display items in a database

  1. Open the database in the Database Designer.

  2. On the Database menu, click Properties to open the Database Properties dialog box.

  3. In the Display area, select those items you want to display.

To expand or collapse a table in a database

  • In the Database Designer, right-click a table and click Expand or Collapse.

To expand or collapse all tables

  • Right-click the Database Designer and click Expand All or Collapse All.
