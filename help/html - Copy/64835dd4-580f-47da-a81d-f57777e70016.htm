New Remote View Dialog Box
See Also






This dialog box lets you create a new remote view using the View Designer or the Remote View Wizard.

Note

Before you can create a new remote view, you must establish a connection or ODBC data source. To establish a connection, use the Connection Designer.

For more information, see Working with Views.

View Wizard

Opens the Remote View wizard, which creates a remote view.

New View

Opens the Select Connection Or Data Source dialog box so you can select a remote data source to use in your remote view. For more information, see Select Connection or Data Source Dialog Box.
