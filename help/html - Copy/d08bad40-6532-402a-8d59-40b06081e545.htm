This sample uses the SetAllx Foundation Class to allow for programmatic setting of object properties based on extended criteria not available with the native SetAll( ) method.

This sample contains the following classes
Class Library Description

_setallx

_setallx.vcx

Provides wrapper method called SetAllx that adds expanded SetAll capabilities.

For details on all classes available in the foundation classes, see Visual FoxPro Foundation Classes A-Z.
