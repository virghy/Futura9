Do Dialog Box
See Also






Lets you specify a program or form file to run.

This dialog box appears when you select Do from the Program menu.

Shortcut

CTRL+D

Do (text box)

Specifies the program or form file to run. To run the program or form, select a name from the list and choose Do, or double-click the file name.

The program files displayed are those in the current directory with the .app, .fxp, .mpr, .prg, .qpr, or .src extension that matches the type specified in the Files of Type box.

Do (button)

Sets the default folder to the one that contains the selected program or form and runs the program or form.
