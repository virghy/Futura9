How to: Add Databases to Projects
See Also 






You can add databases to a project for an application.

To add a database to a project

  1. Open the project in the Project Manager Window.

  2. In the Project Manager, expand the Data node.

  3. Click the Databases node, and then Add.

  4. In the Select database name dialog box, browse to and select the database (.dbc) file for the database you want to add, and click OK.

    The database you select appears in the Databases node of the Project Manager.
