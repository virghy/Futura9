This tab in the Grid builder specifies column captions and the type of control you can add to the Grid.

Note

You must select a column before you can change its heading.

Caption

Specifies the caption for the selected column. Type a caption in the box. If you do not specify a caption, the field names will be the column headings.

Control Type

Specifies the type of control to add to the selected column. Text or edit boxes are available for character data, text or check boxes are available for logical data, and text boxes or spinners are available for numeric data.

Grid Display

Shows the grid columns as currently specified. The display automatically updates according to your Caption and Control Type selections. You can set the columns to the desired width by dragging the right edge of a heading.
