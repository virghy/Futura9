How to: Run Applications
See Also 






After you finish building a redistributable application file, you can run it. You can run application files from the Program menu or by using the DO command. For more information, see DO Command.

To run an application (.app) file

  1. On the Program menu, choose Do.

  2. In the Do dialog box, select the .app file you want to run.

To run an executable (.exe) file

  • In Windows, double-click the icon for the .exe file.

You can also run .exe files from the Program menu or by using the DO command.
