Visual FoxPro XML Web Service Location Dialog Box
See Also






Specifies a local Uniform Resource Locator (URL) or virtual directory for publishing an XML Web service.

The Visual FoxPro XML Web Service Location dialog box appears the first time you select the Web Services Publisher wizard from the Wizards menu. You can open the dialog box by clicking Advanced in the Visual FoxPro XML Web Services Publisher dialog box. For more information, see Visual FoxPro XML Web Services Publisher Dialog Box.

Note

All prior settings for previously published XML Web services are preserved, so you do not have to reset these values each time you want to republish a previously published XML Web service.

Existing

Select an existing location in the Select Location (Virtual Directory) box for publishing an XML Web service. This must be a URL location URL (virtual directory) available on an accessible IIS Web server.

New

Specifies a new virtual directory in the Select Location (Virtual Directory) box for publishing an XML Web service and to type a name for the virtual directory in the New Virtual Directory Name box.

Select Location (Virtual Directory)

Specifies the local URL (virtual directory) to publish an XML Web service. The box lists all available virtual directories on your IIS default Web server. If New is selected, the specified URL is used as the root location for the new virtual directory appearing in the New Virtual Directory Name box.

File Output Path

Displays the physical path for the new virtual directory location and is available only when New is selected. Click Ellipsis (...) button to the right of the File Output Path box to select a physical directory for the new virtual directory.

New Virtual Directory Name

Specifies the name of the new virtual directory and is available only when New is selected.
