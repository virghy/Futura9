Coverage Profiler Find Dialog Box

Enables you to search the Coverage log for specific items.

Options

Match Case

Match the case of the search item

Wrap Around

Specifies that a search begun from within a file, continues past EOF through BOF to the current cursor position.

Search

Classes

Search through the classes in the coverage log.

Files

Search through the files in the coverage log.

Source code

Search through the source code in the current display.
