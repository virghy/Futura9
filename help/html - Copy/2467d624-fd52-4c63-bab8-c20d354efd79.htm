How to: Change Detail Band Order
See Also 






You can change the order in which Detail bands appear in the report or label page layout.

Note

If the _REPORTBUILDER system variable is not set to the default Report Builder, you might not be able to change the order of the Detail bands after you add them. For more information, see _REPORTBUILDER System Variable.

To change the order of Detail bands

  1. Open the report or label for editing in the appropriate designer.

  2. On the Report menu, click Optional Bands.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder, the Optional Bands dialog box opens instead. If _REPORTBUILDER is set to a third-party builder, a different dialog box might display. For more information, see Optional Bands Dialog Box.

  3. In the Report Properties dialog box, click the Optional Bands tab.

  4. In the Detail bands area, drag the mover button next to the Detail band you want to the position you want in the list.

For more information, see Optional Bands Tab, Report Properties Dialog Box (Report Builder).
