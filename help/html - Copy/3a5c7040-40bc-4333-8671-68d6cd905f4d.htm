Set Index Order Dialog Box
See Also






Sets the index order for a child table. This dialog box appears when you set the parent table of a relation in the Data Session Window and then click a child table in the Aliases list of the Data Session window if the child table has no index order already set.

Set Index Order List

Lists any predefined index orders, which exist for the selected child table.

No Order Button

Opens the Expression Builder Dialog Box where you can set your own relationship for the child table. You can also double-click on a table index listed in the dialog box to open the ExpressionBuilder dialog box.
