Visual FoxPro 9.0 SP2
Define Window Layout Dialog Box
See Also






Allows you to name and save window and toolbar configurations for reuse.

View Name

Specifies the name of the current window layout.

View

Lists the available window layouts.

Apply

Configures the current windows to the selected window layout.

Add

Adds the currently named layout to the list of available window layouts.

Delete

Deletes the currently select layout from the list of available window layouts.

Rename

Allows you to change the name of the current window layout.
