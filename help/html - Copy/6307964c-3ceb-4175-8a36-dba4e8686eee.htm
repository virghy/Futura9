This dialog box is part of the Web Publishing wizard and makes it possible for you to specify additional characteristics for your Web page. It is displayed from the Step 3, Advanced button. You can specify formatting and objects from cascading style sheets (.css) and add additional HTML items that can include headings, text, or other Web page items and characteristics.

To access the Advanced dialog box

  1. From Tools menu, choose Wizards, and then click Web Publishing.

  2. Navigate through the wizard until you reach Step 3.

  3. Click Advanced.
Cascading style sheet

This control makes it possible to specify or locate a cascading style sheet (.css) file to use for your Web page.

Body

Color    This drop-down box makes it possible for you to specify a color from the list to use for the body background. This option overrides any color specified in the selected data layout or visual style.

Image    This textbox makes it possible for you to specify or locate an image file to display on your Web page.

Copy graphics to HTML output location

This check box makes it possible for you to specify that any graphic files used in your data layout or visual style are copied to the same location as your Web page file.

Save Settings

This button makes it possible for you to keep your visual style changes in a file to use later.

Load Settings

This button makes it possible for you to access a saved layout or to remove a saved layout you no longer want.

Additional HTML Items

This section makes it possible for you to add HTML tags for formatting, text, and references to your Web page.

Add

This button opens the HTML Item dialog box, so you can specify additional HTML items and formatting for your Web page. For more information, see HTML Item Dialog Box.

Edit

This button opens the HTML Item dialog box, so you can edit the selected HTML item for your Web page. For more information, see HTML Item Dialog Box.

Remove

This button makes it possible for you to delete the selected HTML item from your Web page.
