Visual FoxPro 9.0 SP2
Go To Line Dialog Box (Visual FoxPro)
See Also






This dialog box makes it possible for you to move the insertion point to the designated line in a text or program file or in the Command window.

Line Number

Specifies the line number where you want the insertion point to appear.
