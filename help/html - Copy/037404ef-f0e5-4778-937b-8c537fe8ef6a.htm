Connections Dialog Box
See Also






This dialog box allows you to create and maintain database connections. It appears when you select Connections from the Database menu.

Connections in <database>

Lists the connections associated with the database currently opened.

New

Opens the Connection designer, which allows you to create and modify named connections.

Modify

Opens the Connection designer, which allows you to modify the existing database connection.

Delete

Allows you to delete the connection from the database.
