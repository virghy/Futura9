Dynamic Script Preview Dialog Box (Report Builder)
See Also






Enables you to view the code that will be executed at run time to create your dynamic property display.

Note

Any changes you make in this dialog box will not be saved. You can, however, copy the code into a user-defined function and customize the code there.

You can open this dialog box by selecting a condition and clicking Script in the Dynamics tab of the Report Control Properties dialog box.

  • How to: Dynamically Format Report Controls

Font

Opens the Font Dialog Box so that you can specify the font name, style, and size of the code font.

Code Zoom

Opens the code in a separate editing window so you can see the code with Visual FoxPro's syntax color coding.
