Visual FoxPro 9.0 SP2
Code page for the field "field name" doesn't match cursor's code page (Error 2184)






The XMLAdapter Class ToXML Method generates this error if the RespectCursorCP Property is set to true (.T.), the UTF8Encoded Property is set to false (.F.), the UseCodePage Property is set to true (.T.), the XMLField Class CodePage Property is greater than zero and does not match cursor's code page.
