Include File Dialog Box
See Also






Makes it possible for you to specify a header file containing predefined compile-time constants for a user-defined class, form, or form set.

This dialog box appears when you select Include File from the Class or Form menus.

Include File

Specifies a header file containing predefined compile-time constants for a user-defined class, form, or form set. Type the name of the header file in the box, or use the dialog button to select an existing header file.
