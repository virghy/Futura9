Visual FoxPro 9.0 SP2
The Dockable property cannot be enabled possibly due to conflicting properties (Error 2152)






This error message can be generated if a form's Dockable Property is set to true (.T.) and the form is part of a formset. Also, properties which can potentially conflict with docking behavior (ScrollBars Property, Desktop Property, ShowWindow Property) can generate this error.
