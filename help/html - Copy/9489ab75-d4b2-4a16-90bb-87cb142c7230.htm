How to: Delete Data Groups in Reports
See Also 






You can delete data groups, which deletes the Group Header and Group Footer bands, from reports and labels.

Note

If a data group contains items, the designer prompts you to confirm deleting the band.

To delete a data group

  1. Open the report or label in the appropriate designer.

  2. On the Report menu, click Data Grouping.

    The Report Properties dialog box opens.

    Note

    If the _REPORTBUILDER system variable is not set to the default Report Builder or is set to a third-party builder, the Data Grouping dialog box displays or a different dialog box might display. For more information, see _REPORTBUILDER System Variable and Data Grouping Dialog Box.

  3. In the Report Properties dialog box, click the Data Grouping tab if it is not selected.

  4. In the Group nesting order list, select the data group expression you want, and click Remove. When the designer prompts you to confirm deletion, click Yes to confirm deletion.

    The Group Header and Group Footer bands are deleted from the page layout.

For more information, see Data Grouping Tab, Report Properties Dialog Box (Report Builder).
