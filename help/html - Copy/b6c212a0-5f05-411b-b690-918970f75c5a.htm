Gives general information about the class and its associated icons. The dialog box that contains this tab appears when you choose Class Info from the Class menu.

Toolbar Icon

Displays the name of the class toolbar icon. Choose the dialog button to select an icon for the class.

The class toolbar icon is displayed in the Form Controls toolbar in the Form Designer, and in the Class Designer when you populate the Form Controls toolbar with the classes in your class library. The class toolbar icon also appears in the Project container when the class is part of a project.

Container Icon

Displays the name of the class container toolbar icon. Choose the dialog button to select an icon for the class container.

The container icon is displayed only in the Class Browser window.

OLE Public

Specifies that when a program containing the OLEPublic keyword is built through the Project Manager, a custom Automation server is generated. A Globally Unique Identifier, or GUID, is also generated for the program.

Description

Displays comments about the selected class. You can type a new description in the box or edit an existing one.

Class Name

Displays the name of the class selected in the Class Designer.

Parent Class

Displays name of parent class.

Class Library

Displays path and file name of class library.
