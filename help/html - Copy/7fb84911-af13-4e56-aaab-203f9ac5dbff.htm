Default Data Tab, Pane Customization Dialog Box
See Also






The Default Data tab specifies the default data to display if live data cannot be retrieved from the Internet when a pane first displays. This information is optional and is used only if the data is not stored locally. If a pane successfully retrieves data from the Internet but fails on subsequent attempts, the pane uses the data stored in the PaneCache directory, not the default XML data.

Modify

Opens a window to edit the default XML data.

Edit Box

Displays the default XML data.
