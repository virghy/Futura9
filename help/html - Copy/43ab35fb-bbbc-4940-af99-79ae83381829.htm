How to: Navigate Records in Tables
See Also 






You can navigate records when viewing them in a browse window by using scroll bars, arrow keys, or the TAB key. You can also specify a group of records to locate.

To navigate records in a browse window

  1. Open the table in a browse window.

  2. On the Table menu, point to Go to Record, and choose one of the following:

    • To move to the first record, click Top.

    • To move to the last record, click Bottom.

    • To move to the next record, click Next.

    • To move to the previous record, click Previous.

    • To specify a record number, click Record # to open the Go to Record dialog box, type a number, and click OK.

For more information, see Go To Record Dialog Box.

To specify a group of records to locate

  1. Open the table in a browse window.

  2. On the Table menu, point to Go to Record, and click Locate.

    The Locate dialog box opens.

  3. In the Scope box of the Locate dialog box, select the range of records you want to locate.

  4. In the For box, type an expression that records must meet to be located. To build an expression, click the ellipsis (...) button.

  5. In the While box, type an expression that evaluates to true in order to continue locating records.

  6. When you are finished, click Locate.
