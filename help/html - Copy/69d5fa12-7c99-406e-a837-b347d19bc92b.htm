Visual FoxPro 9.0 SP2
Label file is invalid (Error 54)






An attempt was made to modify or print labels using a label file that was not created with CREATE LABEL or MODIFY LABEL, or that had other label file definition errors.

Note   Label files (lbx) in Visual FoxPro have the same table structure and requirements as Report forms (frx). For examples of report and layout file definition errors, see Report file "name" is invalid (Error 50).
