Debug Output Window
See Also






Makes it possible for you to display output from active programs, procedures, or method code.

This window is activated when you choose Output from the Window Menu, Debugger Window.

Output Pane

Displays character strings specified by the DEBUGOUT command and, if event tracking is enabled, the names of system events as they occur.

To save the text in the window

  • Choose Save Output As from the File menu in the Debugger window.
