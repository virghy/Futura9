Report Designer event handling

Specifies how the Report Builder will respond to events in the Report Designer.

Handler Registry Table

Specifies whether or not the Report Builder uses its internal lookup table or an external, alternative customizable table. The textbox will contain the fully qualified file name of the current report builder event handler registry table. Click the ellipsis (…) to display a file chooser dialog box to select an alternate event handler registry table. This setting will not persist between sessions of Visual FoxPro.

Create copy

Prompts for a file name and location into which will be placed a copy of the default, internal event handler registry table, that is ready for customization.

Registry Explorer

Displays the Event Handler Registry Dialog Box (Report Builder).

Close

Closes the Report Builder Options dialog box.
