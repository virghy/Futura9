How to: Arrange Tables and Views in Databases
See Also 






You can change the layout of tables or views in the Database Designer. For example, after working in the database, you might want to return tables to the default height and width or you might want align the tables.

To arrange tables or views in a database

  1. Open the database in the Database Designer.

  2. On the Database menu, click Arrange.

  3. In the Arrange Tables and Views dialog box, choose the options that you want, and click OK.

For more information, see Arrange Tables and Views Dialog Box.
