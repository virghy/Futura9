Component Gallery Window
See Also






The Component Gallery window displays, as shortcut icons, components available to your Visual FoxPro projects, applications, designers, wizards, and builders, enabling you to manage, in various ways, any file available to your system.

The Component Gallery window contains a Component Gallery Toolbar plus the following options:

Catalog (Left) Pane

Displays the Component Gallery hierarchy of catalogs and folders. Select a catalog or folder here to display the items of the container in the Object panel to the right.

Object (Right) Pane

Displays items contained in a catalog or folder.

For information on using items in the Component Gallery, see Using the Component Gallery.

Description Box

Provides information on the selected item. The Visual FoxPro status bar reports Component Gallery process status.
