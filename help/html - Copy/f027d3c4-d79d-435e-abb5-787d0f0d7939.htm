Visual FoxPro 9.0 SP2
Table Designer (Visual FoxPro)






Use the Table Designer to create and modify database tables, free tables, fields, and indexes. The Table Designer helps you implement advanced features, such as validation rules, default values, and default control classes.

In This Section

Fields Tab, Table Designer

Displays table fields in a scrollable grid.

Indexes Tab, Table Designer

Contains a scrollable grid for defining indexes.

Table Tab, Table Designer

Displays the information about the table and makes it possible for you to specify triggers and record-level rules.
