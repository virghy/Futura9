Visual FoxPro 9.0 SP2
How to: Save Programs
See Also 






After you create or modify a program, save your changes.

To save a program

  • On the File menu, click Save.

Visual FoxPro saves programs with the .prg file name extension. After you save your program, you can run or modify it.

Note

If you save a program that does not have a name, Visual FoxPro displays the Save As dialog box so you can specify a name for the program. If you save a program created in the Project Manager, Visual FoxPro adds the program to the project.
