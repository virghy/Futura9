Export Options Dialog Box
See Also






Specifies which records and fields to export. This dialog box is displayed when you choose Options in the Export dialog box. For more information, see Export Dialog Box.

Criteria

Specifies options that control which records and fields to export and the conditions under which they export.

Scope

Displays the Scope dialog box so you can specify which records to export. For more information, see Scope Dialog Box.

For

Displays the Expression Builder dialog box so you can create a logical expression that that must be true for every record to export. For more information, see Expression Builder Dialog Box.

While

Displays the Expression Builder dialog box so you can create a logical expression that must be true to continue exporting records. For more information, see Expression Builder Dialog Box

Fields

Displays the Field Picker dialog box so you can choose fields to export. For more information, see Field Picker Dialog Box.
