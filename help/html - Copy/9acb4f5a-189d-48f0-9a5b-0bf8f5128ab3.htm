Append From Options Dialog Box
See Also






Specifies fields to append and allows you to build an expression that must be met by records to be appended. This dialog box appears when you choose Options in the Append From Dialog Box.

Criteria

Fields

Displays the Field Picker Dialog Box, where you select fields to append.

For

Displays the Expression Builder Dialog Box, where you create a For expression that every record must meet to be appended.
