Visual FoxPro 9.0 SP2
XMLAdapter XMLName property is an XPath expression (Error 2180)






  • The XMLAdapter Class ToXML Method generates this error if the XMLNameIsXPath Property is set to true (.T.) and the XMLName Property is not empty.

  • The XMLAdapter Class ApplyDiffgram Method and the ChangesToCursor Method generate this error if the XMLNameIsXPath Property is set to true (.T.).

  • The XMLTable Class ToCursor Method generates this error if both the XMLAdapter Class XMLNameIsXPath Property and the IsDiffGram Property are set to true (.T.).
