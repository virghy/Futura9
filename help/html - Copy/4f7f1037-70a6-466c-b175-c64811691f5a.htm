Visual FoxPro 9.0 SP2
Invalid call issued while executing a SQLTABLES( ) sequence (Error 1476)






An invalid SQL pass-through (SPT) function call has been made while a SQLTABLES( ) call was expected to continue the SPT call sequence in progress.

Call SQLCANCEL( ) to cancel the SPT sequence or make valid SPT calls to finish the call sequence.
