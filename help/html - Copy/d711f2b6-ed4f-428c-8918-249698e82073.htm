Visual FoxPro 9.0 SP2
TabStyle Property
See Also 
 






Specifies whether the Page tabs in a page frame are justified or non-justified. Available at design time and run time.


PageFrame.TabStyle[ = nStyle]

Return Value

nStyle

One of the following:

nStyle Description

0

(Default) Justified. The width of each page tab is adjusted to accommodate its caption. If necessary, the width of each page tab is increased so the page tabs span the entire width of the PageFrame.

This setting is ignored if the TabStretch property is set to 1, Single Row.

1

Non-justified. The width of each page tab is not adjusted to accommodate its caption. The width of each page tab is not adjusted so that the page tabs span the entire width of the PageFrame.

This setting is ignored if the TabStretch property is set to 1, Single Row.

Remarks

Applies To: PageFrame
