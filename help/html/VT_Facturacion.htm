Acceso: Facturacion - Movimientos - Facturacion
 






En este formulario se registran las facturas en general.

Campo Descripción

Nro Servicio

Corresponde al Nro. de Servicio.

Esto se utiliza para el caso de que se cuente con el modulo de servicios.
Fecha Fecha de la factura
Número Representa el número de factura o comprobante utilizado.
El campo que aparece deshabilitado corresponde al número interno asignado por el sistema para la factura.
Proyecto Representa en número de proyecto asociado a la factura.
Esto se utiliza cuando se cuenta con el módulo de proyectos.
Sucursal Seleccione la sucursal en donde se esta realizando la facturación.

Atencion: El depósito de donde será descargada la mercadería está asociada a la sucursal. Es decir, cada sucursal posee un depósito predeterminado.
Cliente Indica el código del cliente. Presione F8 para ver la lista de clientes.
RUC Indica el RUC del cliente
Dirección Indica la dirección del cliente, si esta cargado. Esta opcion puede modificar pero la información será registrada en esta factura solamente, no afectará a los datos del clientes.
Tipo Factura Seleccione el tipo de factura o comprobante utilizado.
